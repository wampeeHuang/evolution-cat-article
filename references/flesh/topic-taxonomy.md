# 内容基因分类库

> 自动判定用。遇无法分类的选题 → 报未知 + 建议新分类 → 用户确认 → 新增条目。
> 每分类需 ≥3 篇示例才算稳定。置信度 <0.5 → 必须问用户。

---

## 🛠️ 推荐流

**触发**：推荐工具/产品/资源，目的"推荐用起来"
**读者期待**："这个东西有意思，我也去看看"
**输出**：800-1500 字，朋友安利语气，不塞方法论

**已知示例**：
- stop-slop 开源写作规则集介绍
- n8n 自动化部署指南
- 微信文章导出工具

---

## 📖 复盘流

**触发**：个人实践/试验/踩坑/思考，有第一人称经历
**读者期待**："这个人试过了，结论是..."
**输出**：1200-1800 字，1 个洞察 + 1 个行动建议

**已知示例**：
- HTML 转 PDF 踩坑复盘
- 三连图导出 bug 根因分析
- 飞书到微信的管线迁移记录

---

## 📡 快评流

**触发**：热点事件/新发布/行业动态，时效性强
**读者期待**："发生什么了 + 为什么值得关注"
**输出**：800-1200 字，第一句甩结论

**已知示例**：
- Claude 新版发布解读
- GitHub Trending 周报
- 周刊风格选题摘要

---

## 🎯 主张流

**触发**：有一个核心论断需要多领域证据链支撑。不是"介绍一个东西"，是"说服你一件事"。素材含 ≥2 个来自独立领域的证据指向同一结论。

**读者期待**："看完我信了，而且知道该怎么用"

**输出**：1500-3000 字，总分总结构。开头给核心区分 → 中间故事线×证据线交织 → 结尾新姿势+行动指令。

**已知示例**：
- AI 分不清推理和编造——别把它当知识引擎
- （待累积 ≥3 篇）

---

## 🏗️ 深度流

**触发**：Agent 在 Step 0 自行判定深度潜力 + 素材含 ≥2 个意外发现 + ≥3 个可独立成篇的子论点。三个条件全满足才走深度。

**Agent 判定深度潜力的依据**（满足 ≥2 条即可）：
① 跨领域连接——主题涉及 ≥2 个独立领域，连接点本身构成洞察
② 可展开的理论框架——不是"一个方法"，是"一套可迁移的思维框架"
③ 反直觉对比/数据——有数字或对比挑战常识，值得展开论证

**默认走轻**。好框架用复盘流分享即可——往轻走，不往重走。Agent 判深度 → 告知用户后走深度管线。
**读者期待**："这个框架我帮你拆开揉碎了，看完能用"
**输出**：2000-5000 字，系统化方法论

**已知示例**：
- Karpathy LLM 方法论完整拆解
- stop-slop 写作规则的体系化本土化

---

## 判定流程

```
读输入 → LLM 分类

  置信度 ≥ 0.8 → 直接路由
  置信度 0.5-0.8 → 报 Top 2 候选 + 理由 → 问用户
  置信度 < 0.5 或完全陌生 → 报未知 + 建议新分类名 → 用户确认 → 写入本文件
```
